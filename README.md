## Build from scratch
I supplied the node modules, so it's not necessary to run the commands (just open index.html).
I provided the commands if you want to build from scratch:
```
npm install express
npm install cors
npm install mongoose
npm install faker

node create_db.js
node app.js

open index.html
```
